# ProfeDesk

Aplicación web para gestión de clases — asistencia, seguimiento pedagógico, horarios y más.

Desarrollada como single-file HTML con localStorage. No requiere servidor ni base de datos.

## Acceso

👉 [Abrir ProfeDesk](https://cristianceg77-cloud.github.io/profedesk-v2/)

## Características

- Registro de asistencia con letras de calificación personalizables
- Seguimiento pedagógico por indicadores
- Vista combinada asistencia + seguimiento
- Mapa del aula interactivo
- Horario semanal
- Trayectoria de alumnos
- Backup/restauración de datos en JSON
- Sincronización con Google Drive
- PWA instalable en móvil
