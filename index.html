☁️ Configurar Google Drive
Conectá ProfeDesk con tu Google Drive para guardar y restaurar datos automáticamente desde cualquier dispositivo. Es gratis y tarda 5 minutos.
1
Abrí Google Cloud Console Entrá a console.cloud.google.com → "Nuevo proyecto" → Nombre: ProfeDesk → Crear
2
Habilitá Google Drive API Menú → APIs y servicios → Biblioteca → Buscá "Google Drive API" → Habilitar
3
Creá credenciales OAuth APIs y servicios → Credenciales → + Crear credenciales → ID de cliente OAuth → Aplicación web
4
Agregá el origen autorizado En "Orígenes autorizados de JavaScript" agregá exactamente:
5
Pegá tu Client ID acá abajo Copiá el ID que termina en .apps.googleusercontent.com
Se guarda solo en este dispositivo (localStorage). Nunca se envía a ningún servidor.
